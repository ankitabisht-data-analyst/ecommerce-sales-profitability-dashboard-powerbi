# 📁 Data Folder

| File | Description | Rows | Key Columns |
|---|---|---|---|
| `Orders.csv` | Order-level customer & location data | 500 | Order ID, Order Date, CustomerName, State, City |
| `Details.csv` | Order-level financial & product data | 1,500 | Order ID, Amount, Profit, Quantity, Category, Sub-Category, PaymentMode |

Both files are joined on `Order ID` to build the unified sales model used in Power BI.
